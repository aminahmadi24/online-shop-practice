# Online Shop Practice

A simple practice e-commerce application in Java that demonstrates core online-shopping functionality including product browsing, cart processing, price calculation, and inventory management.

## Project Overview

This project is a **Java-based online shop practice application** developed as a learning exercise. The application implements a basic e-commerce workflow where users can:

- Browse available products
- Add items to a shopping cart
- Calculate total cart price
- Update product inventory when items are purchased
- Process cart checkout

## Features

- Product catalog and listing
- Shopping cart functionality
- Total price calculation for cart items
- Inventory deduction after checkout
- Java + Maven project structure
